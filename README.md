# 🕹️ Retro & Arcade Games Collection

Welcome to my mini-games portfolio! This repository features a collection of interactive games developed to strengthen my programming logic, object-oriented concepts, and GUI development skills using **Java** and **Python**.

---

## 🚀 Featured Projects

Here is a quick overview of the games included in this collection. Each project contains its own detailed source code and execution instructions.

| Game Name | Language / Framework | Type | Description |
| :--- | :--- | :--- | :--- |
| **🎮 Flappy Bird** | Python (Pygame) | 2D GUI Arcade | A classic replica of Flappy Bird with real-time collision detection and physics. |
| **🕹️ Pacman Project** | Java | Console / Terminal | A terminal-based implementation of the classic arcade maze-game. |
| **🔤 Hangman Project** | Java | Console / Terminal | Word guessing game featuring file handling/pre-defined vocabulary lists. |
| **🔢 Number Guessing** | Java | Console / Terminal | A dynamic binary-search-like logic game with user attempts tracking. |
