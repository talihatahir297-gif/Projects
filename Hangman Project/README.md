# Hangman Game (Java)

A simple command-line **Hangman Game** developed in **Java**. The player guesses letters to reveal a hidden word before running out of attempts. This project demonstrates basic Java programming concepts such as loops, conditions, arrays, methods, and user input.

---

## Project Overview

This project is a console-based implementation of the classic Hangman game. It provides an interactive gameplay experience where users guess the hidden word one letter at a time.

---

## Features

- Console-based gameplay
- Random word guessing
- Letter validation
- Limited number of attempts
- Win/Lose result display
- Easy-to-understand Java code

---

## 🛠 Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Java Scanner Class

##  How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/talihatahir297-gif/Hangman.git
```

### 2. Open the Project Folder

```bash
cd Hangman
```

### 3. Compile the Program

```bash
javac Hangman.java hang.java
```

### 4. Run the Program

```bash
java Hangman
```

---

## Gameplay

- The program selects a hidden word.
- Enter one letter at a time.
- Correct letters are revealed.
- Incorrect guesses reduce your remaining attempts.
- Guess the complete word before your attempts run out.
