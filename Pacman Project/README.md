#  Pacman Game (Java)

A simple **Pacman Game** developed in **Java**. The game allows the player to control Pacman, collect points, avoid obstacles, and enjoy a classic arcade-style gaming experience. This project demonstrates Java programming, object-oriented concepts, and game development fundamentals.

---

##  Project Overview

This project is a Java implementation of the classic Pacman game. It focuses on player movement, collision detection, score management, and game logic while providing an interactive gameplay experience.

---

## Features

- Smooth Pacman movement
- Score tracking system
- Collision detection
- Object-oriented design
- Keyboard controls
- Simple and interactive gameplay

---

##  Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Java Swing / AWT
- File Handling (if used)

##  Project Structure

```
Pacman Project/
│── Objects/                     # Game assets/resources

│── Pacman.java                  # Pacman class
│── PacmanGame.java              # Main game logic
│── PacmanGame.class             # Compiled game class
│── PacmanGame$block.class       # Internal compiled class
│── FileHandling.code-workspace  # VS Code workspace
└── README.md                    # Project documentation
```

##  How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/talihatahir297-gif/Pacman-Project.git
```

### 2. Open the Project Folder

```bash
cd Pacman-Project
```

### 3. Compile the Java Files

```bash
javac *.java
```

### 4. Run the Game

```bash
java App
```
