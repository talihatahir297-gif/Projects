#  Flappy Game

A simple Flappy Bird clone developed in **Python** using the **Pygame** library. The game allows the player to control a bird, avoid obstacles, and achieve the highest possible score. A high score system is also included.

---

##  Project Overview

This project is a basic implementation of the popular Flappy Bird game. It demonstrates the use of Python programming, object movement, collision detection, scoring, and file handling.

---

##  Features

- Smooth bird movement
- Randomly generated pipes
- Collision detection
- Score counter
- High score saving
- Simple and user-friendly gameplay

---

## 🛠 Technologies Used

- Python 3.x
- Pygame Library

## 🚀 How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/talihatahir297-gif/Flappygame.git
```

### 2. Open the Project Folder

```bash
cd Flappygame
```

### 3. Install Pygame

```bash
pip install pygame
```

### 4. Run the Game

```bash
python flappygame.py
```
